# 6502-Emulator-Project

This project is inspired by the TokyoEdTech 6502 Emulator Project.  
My goal is to deepen my understanding of computer architecture by building and modifying a basic 6502 CPU emulator in Python. While also tracking changes, errors made, and progress throughout the process.

Current Features
- 64KB memory simulation
- A, X, and Y registers
- Program counter
- Immediate addressing mode
- Loading instructions LDA, LDX, and LDY
- Storing instructions STA,STX, and STY
- Increment and Decrement instructions INX, INY, DEX, DEY
- Transfer instructions TAX, TXA, TAY, TYA
